# Steps to create a JAVA application

1. Write code in editor
2. This code needs to be transformed into another format which can be executed by the computer. This is done by a compiler. Every compiler is specific to a language.
3. The file produced by a compiler is called a binary file or an executable file. This compiled code contains special binary codes called *byte code*.
4. Compilation and execution require two specific pieces of software that are part of the Java Development Kit.

### Convention to name the files:
- The name of a Java file should start with a capital letter.
- The first character needs to be a letter.

### JAVA Components

- JDK: Java Development Kit- The JDK contains many tools and among them are the ones we will be using to compile and run a Java application.

- JRE: Java Runtime Environment - It is a subset of JDK. 

- J2EE- Java EE/Jakarta EE. All these refer to the Java Enterprise Edition. It is a set of tools and libraries to create enterprise class applications.
